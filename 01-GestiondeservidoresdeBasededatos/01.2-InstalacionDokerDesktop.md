## Creación de un contenedor Docker Desktop.

1. Para verificar que Docker esté funcionando, abrimos Git Bash y escribimos el siguiente comando:
```
docker --version (Deberá aparecer la versión)

docker pull mcr.microsoft.com/mssql/server:2019-latest (Estracción del contenedor del registro)
```

2. Nos vamos al navegador y buscamos la `imagen de Docker para SQL Server`.

3. Buscamos getting-started.
![Base de Datos para Negocios Digitales](/img/18.png "BD.")

4. Buscamos este comando y lo ponemos en GitBash.
```
docker pull docker/getting-started

docker images (tendremos nuestra primera imagen)
```
Verificamos en Docker:
![Base de Datos para Negocios Digitales](/img/19.png "BD.")

5. Nos regresamos a Git:
```
docker run(primeros 4 id de la imagen)
```
Nos vamos a `Docker - Containers`, y aparecera el contenedor.
![Base de Datos para Negocios Digitales](/img/20.png "BD.")

Para verificar los contenedores en la consola:
```
docker container ls -a
```
